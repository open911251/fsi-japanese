FSI 日語訓練工具 聽 → 說 → 反覆操練

FSI 口語操練法：先聽不看字、跟讀、代換、應答、逐段累加。搭配 Google Cloud TTS 日語語音。
如何取得 API Key？
到 console.cloud.google.com 建立專案 → 搜尋並啟用「Cloud Text-to-Speech API」→ 「憑證」→「建立憑證」→「API 金鑰」，貼到上方即可。每月 WaveNet/Neural2 有 100 萬字元免費額度，個人練習幾乎用不完。Key 只存在你的瀏覽器（localStorage），不會上傳。
本地 AI 設定（選用：正誤回饋／開放應答）
接自己工作站的 Whisper／LLM（OpenAI 相容 API）。遠端機器用 ssh -L 埠轉發到 localhost 即可，瀏覽器允許 HTTPS 頁面連 localhost。伺服器需允許 CORS（Ollama 設 OLLAMA_ORIGINS=*；faster-whisper 可用 speaches 等 server）。設定只存在瀏覽器。
請按「開始」
錄音對比　聽示範 → 錄自己 → A/B 對比腔調
自訂教材　貼上任何句子（例如大家的日本語課文），每行一句，格式：日文｜中文（中文可省略）
載入後可用「聽讀」與「錄音對比」模式訓練。內容會存在瀏覽器。
